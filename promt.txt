# Prompt

You are a **Staff Software Engineer** and **Principal System Architect** at Google with expertise in React, Progressive Web Apps (PWAs), AI, Retrieval-Augmented Generation (RAG), LLMs, Chrome Apps, and scalable distributed systems.

Design a production-ready application from scratch.

---

# Project

Design a **Flashcard Revision Web Application** similar to **Inshorts**, but for interview preparation.

The application should be installable as a **Progressive Web App (PWA)** and work like a Chrome App.

Users continuously swipe through concise flashcards while revising interview topics.

---

# Functional Requirements

## 1. AI Flashcard Generation

Users provide a **HelloInterview URL**.

Example:

[https://www.hellointerview.com/learn/system-design/in-a-hurry/introduction](https://www.hellointerview.com/learn/system-design/in-a-hurry/introduction)

The application should

* scrape or extract the content
* identify headings
* summarize intelligently
* generate revision-oriented flashcards

Each flashcard should contain

* concise explanation
* key takeaways
* interview tips
* common mistakes
* memorable analogy if applicable

Cards should not be paragraph-heavy.

Target reading time:

15–30 seconds.
